# Screenshots

Live map screenshots captured during system operation.

## Naming convention

| Filename pattern | Contents |
|-----------------|---------|
| `map-overview-YYYYMMDD.png` | Full globe view with all layers active |
| `map-anomaly-overlay-YYYYMMDD.png` | Anomaly markers visible over aircraft |
| `map-pattern-overlay-YYYYMMDD.png` | DBSCAN pattern markers visible |
| `detail-panel-explanation-YYYYMMDD.png` | DetailPanel open with Claude explanation |
| `intel-panel-query-YYYYMMDD.png` | IntelPanel with active NL query result |
| `intel-panel-summary-YYYYMMDD.png` | IntelPanel showing Situation Summary |

## Portfolio usage

At minimum, capture:
1. `map-anomaly-overlay` — the primary "hero" image for the card
2. `detail-panel-explanation` — shows the Claude integration in action
3. `map-overview` — establishes scale (8,000+ aircraft)
